🎓 Student Management System (Django)
📌 Overview

A web-based Student Management System built using Django.
It allows users to manage student records with CRUD operations and authentication.

🚀 Features
🔐 User Registration & Login
➕ Add Student Details
✏️ Edit & Update Records
❌ Delete Students
🔍 Search Functionality
🖼️ Image Upload Support

🛠️ Tech Stack
Python
Django
HTML, CSS
SQLite

⚙️ Installation
git clone https://github.com/psrisaivarsha/Student-management-system.git
cd Student-management-system
pip install -r requirements.txt
python manage.py runserver
